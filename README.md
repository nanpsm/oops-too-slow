## How to Play

1. Enter your nickname  
2. Choose **Solo** or **Team** mode  
3. React to on-screen prompts as fast as possible  
4. Complete all rounds with the lowest time  
---

## Controls

- **Keyboard**: Press the displayed key  
- **Mouse**: Click the target circle  
- **Gesture**: Show the correct hand sign to the camera  
---

## Game Modes

### Solo Mode
- Play alone
- Beat your best time

### Team Mode
- Create or join a room
- Compete on a shared leaderboard
